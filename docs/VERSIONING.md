# Versioning

yllmlog uses date-based versioning with period separators.

Default format:

```text
YY.MM.DD
```

If more than one version is produced on the same day, add a fourth sequence number:

```text
YY.MM.DD.NN
```

Release milestones append `-Release`.

Examples:

```text
26.06.05
26.06.05.02
26.06.05-Release
26.06.05.02-Release
```
