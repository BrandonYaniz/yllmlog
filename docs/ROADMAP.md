# Roadmap

## Initial capability set

- Daemon plus CLI.
- Local socket communication.
- Minimal config file.
- SQLite runtime state.
- Watched logs and glob patterns.
- Offset tracking.
- Rotation detection.
- Rule engine.
- Event grouping.
- Local LLM analysis through yllmd.
- Immediate, daily, and weekly reports.
- Chat-based configuration.

## Future supervised correction capability

- Read-only diagnostics.
- Proposed fixes.
- File backups.
- Diffs.
- Admin approvals.
- Reversible file modifications.
- Restore workflow.
